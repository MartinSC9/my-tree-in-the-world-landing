# Documentación por Rol

Esta carpeta contiene la documentación específica para cada tipo de usuario de la plataforma.

## Roles Disponibles

| Carpeta | Rol | Descripción |
|---------|-----|-------------|
| [01-usuario](./01-usuario/) | Usuario Final | Personas que compran árboles individuales o participan en proyectos colaborativos |
| [02-empresa](./02-empresa/) | Empresa | Empresas que compensan huella de carbono y crean proyectos colaborativos |
| [03-vivero](./03-vivero/) | Vivero | Proveedores que preparan y venden árboles |
| [04-plantador](./04-plantador/) | Plantador | Trabajadores que plantan los árboles físicamente |
| [05-admin](./05-admin/) | Administrador | Gestión y supervisión de toda la plataforma |
